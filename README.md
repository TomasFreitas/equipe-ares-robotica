# 🤖 Equipe Ares — Site Oficial

> Site oficial da Equipe Ares de Robótica do IFPB Campus Cajazeiras, desenvolvido como projeto da disciplina de Engenharia de Software.

---

## 🎯 Objetivo

Este repositório contém o código-fonte do site da **Equipe Ares**, time de robótica estudantil do Instituto Federal da Paraíba (IFPB), Campus Cajazeiras. O site apresenta a história, conquistas e integrantes da equipe, além de servir como vitrine para patrocinadores e parceiros.

## ⚙️ Fluxo de Trabalho Git

Este projeto segue o **GitHub Flow**:

1. A branch `main` é protegida — nenhum commit direto permitido
2. Cada funcionalidade é desenvolvida em uma branch separada (`feature/nome-da-feature`)
3. Ao finalizar, abre-se um **Pull Request** para `main`
4. É necessária ao menos **1 aprovação** (code review) para mesclar
5. Após aprovação, o PR é mesclado e a branch de feature pode ser deletada
